# Download Novara ISO (No Commands)

If you do not want to run any commands, use this direct GitHub release asset:

- **Direct ISO download:**
  `https://github.com/<OWNER>/<REPO>/releases/latest/download/Novara%20Intel-amd%20x64.iso`
- **Checksum:**
  `https://github.com/<OWNER>/<REPO>/releases/latest/download/Novara%20Intel-amd%20x64.iso.sha256`

## VMware Workstation Pro settings (recommended)

- Guest OS Family: **Linux**
- Guest OS Version: **Debian 12.x 64-bit**
- vCPU: **4 cores**
- RAM: **6144 MB (6 GB)**
- Disk: **40 GB**
- Firmware: **UEFI**
- 3D Acceleration: **Enabled**

## Install flow

1. Click the ISO link above.
2. Open VMware Workstation Pro.
3. Create a new VM and choose that downloaded `.iso` file.
4. Use the recommended VM settings listed above.
5. Boot and install.
